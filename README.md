# Inventory Management System
A Web Application for Inventory Management, served as a project for Web Application Development Lab at International University, VNU-HCMC.

## Technology
- Frontend: React
- Backend: Spring Boot
- Database: MySQL

## How to run this project from source
This project comes with `mvnw` and `mvnw.cmd`, which are Maven wrapper generated by Visual Studio Code. Executing one of these wrappers will automatically download Maven to your system.

To run this project, first clone this repo:
```sh
git clone https://github.com/islabre1426/Inventory-Management-System
```

Then, on Windows, run (PowerShell recommended):
```powershell
& mvnw.cmd spring-boot:run
```

If you're on Linux or MacOS, run:
```sh
./mvnw spring-boot:run
```
